## Introduction

A Customer management  website usingReact JS, Node JS, Express JS, and MongoDb.

NOTE: Please read the RUN section before opening an issue.

This website is a customer management website were admin can Add,Edit,Delete,and Get the user details.  Image of the user can be added in this application.

## Run

To run this application, you have to set your own environmental variables. For security reasons, some variables have been hidden from view and used as environmental variables with the help of dotenv package. Below are the variables that you need to set in order to run the application:
MongoURl: should be added in your env file.
cloudinary key adn secret is provided.

After you've set these environmental variables in the .env file at the root of the project, and intsall node modules using  `npm install`

Now you can run `npm start` in the terminal and the application should work.
